# UrbanLens — Urban Data Studio

A responsive urban research and data visualization studio landing page built with HTML, CSS and JavaScript.

## Overview

UrbanLens is a clean editorial website concept for a studio working with urban research, spatial data and information design.  
The layout combines project case studies, journal content and educational sections.

## Features

- Responsive navigation
- Urban research hero section
- Project case study cards
- Academy section
- Editorial journal layout
- About section
- Newsletter form
- Scroll reveal animations
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`
