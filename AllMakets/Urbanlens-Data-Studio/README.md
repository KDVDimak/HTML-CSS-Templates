# UrbanLens — Urban Data Studio

A responsive urban research and data visualization studio landing page built with HTML, CSS and JavaScript.

## Features

- Responsive navigation
- Urban research hero section
- Project case study cards
- Academy section
- Editorial journal layout
- Newsletter form
- Scroll reveal animations
- External image URLs

## Images

All images are loaded from external Unsplash URLs. No local image files are required.

## Files

- `index.html`
- `style.css`
- `script.js`
