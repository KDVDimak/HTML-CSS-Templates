# Élan — Fashion Store

A responsive fashion store landing page built with HTML, CSS and JavaScript.

## Overview

Élan is a modern fashion brand concept focused on clean layouts, editorial styling and product presentation.  
The design uses large visuals, minimal typography and structured content sections.

## Features

- Responsive navigation
- Promotional hero section
- Product categories
- New collection section
- Product cards
- Brand story
- Newsletter form
- Scroll reveal animations
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`

## What I Practiced

- Building ecommerce-inspired layouts
- Creating product card grids
- Working with editorial spacing
- Designing responsive navigation
- Improving visual consistency
