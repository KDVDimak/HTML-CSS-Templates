# Taskly — Freelance Marketplace

A responsive freelance marketplace landing page built with HTML, CSS and JavaScript.

## Overview

Taskly is a modern freelance platform concept for finding specialists, comparing profiles and managing digital work.  
The project focuses on usability, clear navigation and business-oriented interface design.

## Features

- Responsive navigation
- Search interface
- Service categories
- Freelancer profile cards
- Three-step process section
- Business landing section
- Scroll reveal animations
- External image URLs
- Font Awesome icons

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid
- Font Awesome

## Files

- `index.html`
- `style.css`
- `script.js`

## What I Practiced

- Structuring marketplace interfaces
- Creating reusable card components
- Building responsive navigation
- Designing search and service sections
- Adding scroll-based animations
