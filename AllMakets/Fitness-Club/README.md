# Apex — Fitness Club

A responsive fitness club landing page built with HTML, CSS and JavaScript.

## Overview

Apex is a bold and modern fitness website concept designed for a premium training club.  
The page highlights programs, coaches, memberships and recovery services.

## Features

- Responsive navigation
- Hero section
- Training programs
- Coach profiles
- Membership plans
- Fitness schedule
- Call-to-action section
- Scroll reveal animations
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`

## What I Practiced

- Building a high-contrast visual style
- Creating pricing and schedule sections
- Working with responsive content grids
- Designing strong call-to-action blocks
- Adding simple JavaScript interactions
