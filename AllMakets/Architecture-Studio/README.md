# Forma — Architecture Studio

A responsive architecture studio landing page built with HTML, CSS and JavaScript.

## Overview

Forma is a modern website concept for an architecture and interior design studio.  
The project focuses on strong typography, minimal layouts, large visual sections and a clean editorial style.

## Features

- Responsive navigation
- Large hero section
- Project showcase
- Services section
- Studio information
- Journal section
- Contact call-to-action
- Scroll reveal animations
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`

## Images

All images are loaded from external URLs. No local image files are required.

## What I Practiced

- Building editorial page layouts
- Working with large image sections
- Creating responsive grid systems
- Improving visual hierarchy
- Adding simple interface animations
