# Morrow — Coffee Shop

A responsive coffee shop landing page built with HTML, CSS and JavaScript.

## Overview

Morrow is a warm and minimal website concept for a modern neighborhood coffee shop.  
The project combines soft colors, editorial typography and a relaxed visual atmosphere.

## Features

- Responsive navigation
- Hero section
- Coffee menu
- About section
- Featured products
- Journal section
- Opening hours
- Contact section
- Scroll reveal animations
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`

## What I Practiced

- Designing a warm visual identity
- Creating responsive card layouts
- Organizing content sections
- Building mobile navigation
- Adding smooth scrolling and reveal effects
