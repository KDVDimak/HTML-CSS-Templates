# Roamly — Travel Agency

A responsive travel agency landing page built with HTML, CSS and JavaScript.

## Overview

Roamly is a modern travel website concept designed around curated journeys and memorable experiences.  
The project combines large photography, strong typography and simple interaction patterns.

## Features

- Responsive navigation
- Full-screen hero section
- Destination highlights
- Travel services
- Editorial journal section
- Contact call-to-action
- Scroll reveal animations
- Smooth scrolling
- External image URLs

## Built With

- HTML5
- CSS3
- JavaScript
- Flexbox
- CSS Grid

## Files

- `index.html`
- `style.css`
- `script.js`

## What I Practiced

- Building immersive hero sections
- Working with large background images
- Creating responsive layouts
- Improving typography and spacing
- Adding navigation and scroll animations
